CSS3 Анимации

Animation

animation: animation-name
			 animation-duration
			 animation-timing-function
			 animation-delay
			 animation-iteration-count: number | infinite
			 animation-direction
			 animation-fill-mode
			 animation-play-state: running | paused

Порядок выполнения анимации

animation-direction: normal
			 reverse
			 alternate
			 alternate-reverse

Поведение после завершения

animation-fill-mode: none
			 forwards
			 backwards
			 both

Пауза

@keyframes

@keyframes identifier {
			 from { top: 0; }
			 to { top: 0; }
			 }
@keyframes identifier {
			 0% { top: 0; }
			 50% { top: 10px; }
			 100% { top: 0; }
			 }
